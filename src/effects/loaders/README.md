# `/src/effects/loaders` — Indicadores de Carga

Componentes de loading animados corporativos.

## Componentes

| Componente | Props | Descripción |
|---|---|---|
| `LoaderIntegral` | `size`, `color` | Loader con forma de integral matemática animada |

## Uso

```jsx
import { LoaderIntegral } from "../effects/loaders";

{isLoading && <LoaderIntegral size="lg" color="aquaTurquesa" />}
```
