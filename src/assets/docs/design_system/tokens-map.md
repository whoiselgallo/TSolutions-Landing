# TSolutions IPIDD — Tokens Map
Documento oficial del Design System que describe todos los tokens disponibles,
su propósito y cómo deben utilizarse en el proyecto.

## 🎨 Colores Corporativos

| Token | Valor | Uso |
|-------|--------|-----|
| `--color-naranjaEnergy` | #F97316 | Botones primarios, acentos |
| `--color-aquaTurquesa` | #00E5FF | Glows, highlights, elementos interactivos |
| `--color-negroProfundo` | #000000 | Fondos oscuros, modo corporativo |
| `--color-blancoPuro` | #FFFFFF | Fondos claros, tarjetas, inputs |
